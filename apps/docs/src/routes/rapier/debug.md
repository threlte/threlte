Use the Debug component to see live representations of all colliders in a scene.


```svelte
<Canvas>
	<World>
		<Debug />

		<RigidBody>
			<AutoColliders>
				<Mesh {geometry} {material} />
			</AutoColliders>
		</RigidBody>
	</World>
</Canvas>
```

!!!
