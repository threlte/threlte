<Example path="extras/svg" />

## Examples

### Basic Example

```svelte title="Scene.svelte"
<script lang="ts">
  import { T } from '@threlte/core'
  import { SVG } from '@threlte/extras'
</script>

<SVG
  src="/icons/svelte.svg"
  scale={0.005}
  position.x={-1.2}
  position.y={1.5}
/>
```
