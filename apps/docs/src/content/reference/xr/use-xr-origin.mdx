Returns the nearest parent `<XROrigin>` group, or `undefined` when no `<XROrigin>` exists.

```svelte
<script>
  import { useXROrigin } from '@threlte/xr'

  const origin = useXROrigin()

  if (origin.current !== undefined) {
    origin.current.position.set(0, 0, 5)
  }
</script>
```

Primarily useful for libraries or custom hooks that need to reason about whether a user is inside an `<XROrigin>` (e.g. to decide between moving the origin vs. mutating the XR reference space).

### Signature

```ts
// { current: THREE.Group | undefined }
const origin = useXROrigin()
```
