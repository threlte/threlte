`@threlte/sv` is a [Svelte CLI (`sv`)](https://svelte.dev/docs/cli/overview) community add-on that helps you set up Threlte in your SvelteKit projects.

## Installation

To install the add-on in an existing SvelteKit project, run:

```bash
npx sv add @threlte/sv
```

## What You Get

- `@threlte/core` - The core 3D rendering framework
- Optional packages: extras, rapier, theatre, xr, flex, studio, gltf, uikit
- Optional demo page with a 3D rotating cube

## Options

### `demo`

Include a demo page with a 3D rotating cube.

Default: `false`

```bash
npx sv add threlte="demo:false"
```

### `packages`

Which Threlte packages to install. Use multiple times to select multiple packages.

Default: `extras`

```bash
npx sv add threlte="packages:extras,rapier,theatre"
```

## Available Packages

| Package   | Description                         |
| --------- | ----------------------------------- |
| `extras`  | Components, helpers, hooks and more |
| `rapier`  | Physics engine                      |
| `theatre` | Animation library                   |
| `xr`      | VR and AR support                   |
| `flex`    | Flexbox in 3D                       |
| `studio`  | Visual editor                       |
| `gltf`    | CLI tool for GLTF conversion        |
| `uikit`   | UI toolkit for Threlte              |
