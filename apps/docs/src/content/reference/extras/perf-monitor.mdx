Utility component for monitoring performance details using [stats-gl](https://github.com/RenaudRohlinger/stats-gl).

<Example path="extras/perf-monitor" />

There is some optional [renderer info](https://threejs.org/docs/index.html#api/en/renderers/WebGLRenderer.info) available using the `extraInfo` prop.

## Usage

Install `stats-gl` then simply drop in the `<PerfMonitor>` component as a child of your Threlte `<Canvas>`.

```svelte
<Canvas>
  <PerfMonitor />
  <Scene />
</Canvas>
```
